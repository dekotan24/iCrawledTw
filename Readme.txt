/* Twitterメディアコンテンツ取得ツール - iCrawledTw */
/*--------------------------------------------------*/


1. 機能
	・TwitterのURLからメディアURLを取得します。
	・コンテンツのダウンロードも可能です。
	・ユーザ一覧取得、単ツイート取得の両方に対応しています。
	・プロキシを使用可能です。
	・ツイート取得間隔を変更可能です。


2. 準備するもの
	・ffmpeg
		→動画やgifのダウンロードに使用
		※コマンドラインから呼び出せる状態にすること

	・Google Chrome
		→クロールに使用

	・ChromeDriver
		→Chromeの自動制御に使用
		※アプリケーション実行時に自動取得できなかった場合のみ手動配置が必要


3. 使い方
	◇ダウンロード
		a. 「1: メディアURL解析」を選択
		b. ダウンロードしたいユーザのプロフィールページもしくは単ツイートURLを入力
		c. 自動クロールが実行される

	◇設定変更
		a. 「2: 設定画面」を選択
		b. 設定項目がグループ分けされているので、変更したい番号を入力
		c. 設定項目が表示されるので、変更したい番号を入力


4. 開発環境
	Visual Studio 2022 Community
	.NET 8.0-windows10.0.19041.0
